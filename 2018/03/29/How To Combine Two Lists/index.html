Combine The Sequence List

Subject: Write code to realize a function that combine two backward sequence lists to one list.The result is in the first list.(Do not use a new list for temporary storage.)

How to do this?

Suppose there are two sequence lists,one is A,the other is B.We can move all the elements which are store in B to the back of A.After doing this,use a sorting algorithm to change the order in A,that is the result.But this means may a little redundancy.Is there any lightweight algorithm to realize that function?

Thinking: We draw a picture first.We can set two index value i and j,we use i to track the position of La at a time,use j to track the position of Lb.Because we will combine b to a,we can compare La’s element(La.elem[i]) with b’s element(Lb.elem[j]) from the tail to the head.Every time we compare La.elem[i] with Lb.elem[j],if La.elem[i] is smaller than Lb.elem[j],we move all the elements after La.elem[i] first,then move Lb.elem[j]to La.elem[i],and La.length increments.But we should notice that when the i=La.length-1,there’s nothing to be moved,so this case need to be handle Individually.If La.elem[i] is bigger than Lb.elem[j],just move the i point to the previous element.There are three possibilities that cause the cycle finished:One is i is smaller than 0 and j also smaller than 0;Or i is smaller 0 but j>=0;or i>=0 but j<0.We need to deal with these cases.when i,j are all smaller than 0 or when i is bigger or equals 0 and j<0,it indicates that the combine is finished;but when i is smaller 0 but j is bigger or equals 0,it shows that there are some elements in Lb that never been inserted into La,so we have to insert them in the head of La.There is the code:
